Detailed Description

Base class for all threads.

This class encapsulates all the OS-level thread handling. All thread instances are automatically added to the threading::Manager for management. The manager also takes care of deleting them (which must not be done manually).

◆ SignalStop()

void zeek::threading::BasicThread::SignalStop ( )

Signals the thread to prepare for stopping, but doesn't block to wait for that to happen. Use WaitForStop() for that.

The method lets Terminating() now return true, it does however not force the thread to terminate. It's up to the Run() method to to query Terminating() and exit eventually.

Calling this method has no effect if Start() hasn't been executed yet.

Only Zeek's main thread must call this method.

◆ Start()
