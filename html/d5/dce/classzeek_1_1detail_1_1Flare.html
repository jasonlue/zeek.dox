◆ Flare()

zeek::detail::Flare::Flare ( )

Create a flare object that can be used to signal a "ready" status via a file descriptor that may be integrated with select(), poll(), etc. Not thread-safe, but that should only require Fire()/Extinguish() calls to be made mutually exclusive (across all copies of a Flare).
