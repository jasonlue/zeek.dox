◆ MIME_Matches

using zeek::detail::RuleMatcher::MIME_Matches = std::map<int, std::set<std::string>, std::greater<int> >

Data structure containing a set of matching file magic signatures. Ordered from greatest to least strength. Matches of the same strength will be in the set in lexicographic order of the MIME type string.
