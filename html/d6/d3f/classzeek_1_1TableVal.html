Finds an index in the table and returns its associated value.

Parameters

Returns
The value associated with the index. If the index doesn't exist, this is a nullptr. For sets that don't really contain associated values, a placeholder value is returned to differentiate it from non-existent index (nullptr), but otherwise has no meaning in relation to the set's contents.
