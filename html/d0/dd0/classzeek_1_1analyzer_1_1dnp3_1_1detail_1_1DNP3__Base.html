Buffers packet data until it reaches a specified length.

Parameters
endpan endpoint speaking DNP3 to which data will be buffered.
target_lenthe required length of the buffer
datasource buffer to copy bytes from. Will be incremented by the number of bytes copied by this function.
lenthe number of bytes available in data. Will be decremented by the number of bytes copied by this function.
Returns
-1 if invalid input parameters were supplied, 0 if the endpoint's buffer is not yet target_len bytes in size, or 1 the buffer is the required size.
