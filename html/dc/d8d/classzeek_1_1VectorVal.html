◆ Concretize()

bool zeek::VectorVal::Concretize ( const TypePtr &  t)

Ensures that the vector can be used as a "vector of t". In general, this is only relevant for objects that are typed as "vector of any", making sure that each element is in fact of type "t", and is internally represented as such so that this object can be used directly without any special-casing.

Returns true if the object is compatible with "vector of t" (including if it's not a vector-of-any but instead already a vector-of-t), false if not compatible.

Parameters
tThe yield type to concretize to.
Returns
True if the object is compatible with vector-of-t, false if not.
