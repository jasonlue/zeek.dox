Precompiles a filter and associates a given index with it. The filter syntax is defined by the packet source's implenentation.

Derived classes must implement this to implement their filtering. If they want to use BPF but don't support it natively, they can call the corresponding helper method provided by PktSrc.

Parameters
indexThe index to associate with the filter
filterThe filter string to precompile.
Returns
True on success, false if a problem occurred or filtering is not supported.
