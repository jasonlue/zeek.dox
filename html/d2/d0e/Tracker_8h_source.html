3 // C++ compiler helper class that tracks distinct instances of a given key,
4 // where the key can have any IntrusivePtr type. The properties of a
5 // tracker are that it (1) supports a notion that two technically distinct
6 // keys in fact reflect the same underlying object, (2) provides an
7 // instance of such keys to consistently serve as their "representative", and
8 // (3) provides names (suitable for use as C++ variables) for representative
9 // keys.
10 //
11 // Notions of "same" are taken from hash values ala those provided by
12 // ProfileFunc.
