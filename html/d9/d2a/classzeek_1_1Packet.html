Parameters
link_typeThe link type in the form of a DLT_* constant.
tsThe timestamp associated with the packet.
caplenThe number of bytes valid in data.
lenThe wire length of the packet, which must be more or equal caplen (but can't be less).
dataA pointer to the raw packet data, starting with the layer 2 header. The pointer must remain valid for the lifetime of the Packet instance, unless copy is true.
copyIf true, the constructor will make an internal copy of data, so that the caller can release its version.
tagA textual tag to associate with the packet for differentiating the input streams.
