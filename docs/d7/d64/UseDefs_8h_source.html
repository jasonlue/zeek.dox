14// UseDefs track which variables (identifiers) are used at or subsequent
15// to a given (reduced) Statement. They allow us to determine unproductive
16// variable assignments (both to warn the user, and to prune temporaries),
17// and also accesses to globals (so we know which ones need to be synchronized
18// across function calls).
19
