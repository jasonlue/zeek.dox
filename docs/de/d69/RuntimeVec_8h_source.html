3// Run-time support for vector-oriented operations in C++-compiled scripts.
4// The scope is unary (including appending), binary, and conditional
5// operations. It does not include operations common to other aggregates,
6// such as indexing and explicit coercion (but does include low-level
7// coercion needed to support unary and binary operations).

16// Appends v2 to the vector v1. A separate function because of the
17// need to support assignment cascades.
18inline ValPtr vector_append__CPP(VectorValPtr v1, const ValPtr& v2)
19 {
20 v1->Assign(v1->Size(), v2);
