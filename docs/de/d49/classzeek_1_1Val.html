◆ Footprint() [1/2]

unsigned int zeek::Val::Footprint ( ) const
inline

Returns the Val's "footprint", i.e., how many elements / Val objects the value includes, either directly or indirectly. The number is not meant to be precise, but rather comparable: larger footprint correlates with more memory consumption.

Returns
The total footprint.
