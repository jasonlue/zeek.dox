Detailed Description

Analyzes security data of ADAT exchanges over FTP control session (RFC 2228). Currently only the GSI mechanism of GSSAPI AUTH method is understood. The ADAT exchange for GSI is base64 encoded TLS/SSL handshake tokens. This analyzer just decodes the tokens and passes them on to the parent, which must be an SSL analyzer instance.
