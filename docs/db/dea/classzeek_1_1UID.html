Detailed Description

A class for creating/managing UIDs of arbitrary bit-length and converting them to human-readable strings in Base62 format.

Initialize a UID of a given bit-length, optionally from given values.

Parameters
bitsThe desired length in bits of the UID, up to a max of UID_LEN * 64.
vA pointer to an array of values with which to initialize the UID. If empty or doesn't contain enough values to satisfy bits, then values are automatically generated using calculate_unique_id(). If bits isn't evenly divisible by 64, then a value is truncated to bit in desired bit-length.
nnumber of 64-bit elements in array pointed to by v.
