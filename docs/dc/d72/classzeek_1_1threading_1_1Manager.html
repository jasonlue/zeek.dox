Detailed Description

The thread manager coordinates all child threads. Once a BasicThread is instantitated, it gets addedd to the manager, which will delete it later once it has terminated.

In addition to basic threads, the manager also provides additional functionality specific to MsgThread instances. In particular, it polls their outgoing message queue on a regular basis and feeds data sent into the rest of Zeek. It also triggers the regular heartbeats.
