bool zeek::input::Manager::CreateAnalysisStream ( RecordVal *  description)

Creates a new input stream which will forward the data from the data source on to the file analysis framework. The internal BiF defined in input.bif just forward here. For an input reader to be compatible with this method, it must be able to accept a filter of a single string type (i.e. they read a byte stream).

Parameters
descriptionA record of the script type Input::AnalysisDescription

◆ ForceUpdate()

bool zeek::input::Manager::ForceUpdate ( const std::string &  id)

Force update on a input stream. Forces a re-read of the whole input source. Usually used when an input stream is opened in managed mode. Otherwise, this can be used to trigger a input source check before a heartbeat message arrives. May be ignored by the reader.

Parameters
idThe enum value corresponding the input stream.

This method corresponds directly to the internal BiF defined in input.bif, which just forwards here.

bool zeek::input::Manager::IsCompatibleType ( Type *  t,

)
static

Checks if a Zeek type can be used for data reading. Note that this function only applies to input streams; the logging framework has an equivalent function; however we support logging of a wider variety of types (e.g. functions).

Parameters
tThe type to check.
atomic_onlySet to true to forbid non-atomic types (records/sets/vectors).
Returns
True if the type is compatible with the input framework.
