Detailed Description

template<class T>
class zeek::Span< T >

Drop-in replacement for C++20's std::span with dynamic extent only: https://en.cppreference.com/w/cpp/container/span. After upgrading to C++20, this class may get replaced with a type alias instead and/or deprecated.
