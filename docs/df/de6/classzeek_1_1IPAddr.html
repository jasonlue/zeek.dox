◆ Mask()

void zeek::IPAddr::Mask ( int  top_bits_to_keep)

Masks out lower bits of the address.

Parameters
top_bits_to_keepThe number of bits not to mask out, counting from the highest order bit. The value is always interpreted relative to the IPv6 bit width, even if the address is IPv4. That means if compute 192.168.1.2/16, you need to pass in 112 (i.e., 96 + 16). The value must be in the range from 0 to 128.

◆ ReverseMask()

void zeek::IPAddr::ReverseMask ( int  top_bits_to_chop)

Masks out top bits of the address.

Parameters
top_bits_to_chopThe number of bits to mask out, counting from the highest order bit. The value is always interpreted relative to the IPv6 bit width, even if the address is IPv4. So to mask out the first 16 bits of an IPv4 address, pass in 112 (i.e., 96 + 16). The value must be in the range from 0 to 128.
