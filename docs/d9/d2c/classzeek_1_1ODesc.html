Returns the location of the first place in the bytes to be hex-escaped.

Parameters
bytesthe starting memory address to start searching for escapable character.
nthe maximum number of bytes to search.
Returns
a pair whose first element represents a starting memory address to be escaped up to the number of characters indicated by the second element. The first element may be 0 if nothing is to be escaped.
