Detailed Description

Manager class for IO sources. This handles all of the polling of sources in the main loop.

Calls the appropriate poll method to gather a set of IOSources that are ready for processing.

Parameters
readya vector used to return the ready sources.
timeoutthe value to be used for the timeout of the poll. This should be a value relative to the current network time, not an absolute time value. This may be zero to cause an infinite timeout or -1 to force a very short timeout.
timeout_srcThe source associated with the current timeout value. This is typically a timer manager object.

void zeek::iosource::Manager::Register ( IOSource *  src,
bool  dont_count = false,
bool  manage_lifetime = true

Registers an IOSource with the manager. If the source is already registered, the method will update its dont_count value but not do anything else.

Parameters
srcThe source. The manager takes ownership.
dont_countIf true, this source does not contribute to the number of IOSources returned by Size(). The effect is that if all sources except for the non-counting ones have gone dry, processing will shut down.
