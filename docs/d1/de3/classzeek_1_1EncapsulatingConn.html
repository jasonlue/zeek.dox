Detailed Description

Represents various types of tunnel "connections", that is, a pair of endpoints whose communication encapsulates inner IP packets. This could mean IP packets nested inside IP packets or IP packets nested inside a transport layer protocol. EncapsulatingConn's are assigned a UID, which can be shared with Connection's in the case the tunnel uses a transport-layer.
