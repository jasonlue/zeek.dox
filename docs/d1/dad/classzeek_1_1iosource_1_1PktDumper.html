◆ Close()

virtual void zeek::iosource::PktDumper::Close ( )
pure virtual

Called by the manager system to close the dumper.

Derived classes must implement this method. If successful, the implementation must call Closed(); if not, it must call Error() with a corresponding message.

Implemented in zeek::iosource::pcap::PcapDumper.
